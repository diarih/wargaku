# Dashboard Roadmap

## MVP (implemented)

- Summary cards: total KK, total warga, data baru bulan ini, data belum lengkap.
- Quick action shortcuts.
- Latest resident activity feed.
- Household list and household detail view.

## Next Iteration

- Global search by No KK, NIK, and resident name.
- Create/edit forms for household and resident.
- Data completeness score per household.
- Upload and manage household/resident files.

## Future Enhancements

- Geographic filters (kelurahan/kecamatan).
- Export CSV report.
- Bulk import template.
- Audit timeline per record.
