# Page Checklist

Use this checklist before finishing any new page.

- [ ] Page has clear title and one-line context.
- [ ] Main CTA is visible without scrolling on desktop.
- [ ] Mobile layout works at `360px` width.
- [ ] Loading state exists for async data.
- [ ] Empty state exists when data is not available.
- [ ] Error feedback is visible and actionable.
- [ ] Form fields have labels and validation messages.
- [ ] Success/failure actions trigger toast feedback.
- [ ] Colors, radius, and spacing use design tokens from `globals.css`.
- [ ] No raw inline styles unless technically required.
